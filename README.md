# Nuxt Marketplace

Multi-vendor marketplace with product listings, Pinia cart store, category and price filters, and server-side listing API.

**Stack:** Nuxt | **Complexity:** complex
**Tags:** `nuxt`, `vue`, `marketplace`, `ecommerce`, `pinia`, `shop`, `listings`, `typescript`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
